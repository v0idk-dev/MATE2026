# MATE2026 Fish & Chips Technologies Software Respository for FIH

This repository contains the following components:

## Main Components

### [app](/app/)

**The frontend, computer based system**

The integrated control system app used to operate FIH.

### [control](/control/)

**The backend, onboard control system**

Files used to integrate FIH's control system to a computer, in addition to the RobotController module for the robot's onboatd system.

## Supporting Components

### [camera-calibration](/camera-calibration/)

Files used to calibrate FIH's stereo vision cameras to provide essential intrinsic and extrinsic calibration files for the process of the photogrammetry task.

### [photogrammetry](/photogrammetry/)

All files used in the production photogrammetry system present in the app. Check [in the app folder](/app/scripts/task1_2/) to find the latest version.

### [crab-detect](/crab-detect/)

Location that stores all training data (linked to LFS), annotations, all outputted files, and training files to train on any computer.

### [threat-level](/threat-level/)

Core files used for the iceberg threat level calculator in the integrated app.

## Miscellaneous

### [backups](/backups/)

Used to store backups for core files located within the integrated app.

### [appdesign.jpg](/appdesign.jpg)

Hand-drawn blueprint for the core design of the app.
